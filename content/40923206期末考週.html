40923206期末考週

(1).利用 老師連結 目錄中的 uArm 機械手臂零件, 修改 "nx12絞鍊"與 "front joint" 零件的外型, 使能將 "Triangular connecting rod", "left link" 與 "end link" 等零組件, 與 "right crank" 與 "right link" 的組件左右互換.

(2). 完成題目 1. 的任務後, 在個人 cd2022 網頁中, 新增一個 H1 格式標題為 "期末考週" 的頁面, 並將題目 1. 的零組件繪圖過程, 以 Wink 影片展示過程細節說明, 並在頁面中說明所採用的 NX 套件版本與題目 1. 的任務執行心得.

心得：本次採用NX12來繪製修改零件，是運用老師之零件檔的草圖進行修改，在修改nx12絞鍊時，因為是左右相反，所以有些草圖需要重新繪製，一邊看著原先的零件草圖一邊計算去完成繪製。

40923206-期末考NX檔案.zip

front joint修改影片

nx12絞鍊修改影片

3. 請在完成新幾何外型組立的 uArm 機械手臂後, 將組立件轉入 CoppeliaSim 4.3.0 rev3. 後, 套用 <https://www.coppeliarobotics.com/helpFiles/en/simUI-widgets.htm 圖形介面, 以 slider 控制前三軸的轉動. 完成後, 請在個人網頁的 "期末考週" 頁面, 加入第二個 Wink 影片展示過程細節說明.

4. 完成新幾何外型組立的 CoppeliaSim 場景後, 請與同組組員合作, 透過 https://mde.tw/cd2022_guide/content/Visualization%20Stream.html 模式, 在 Teams 會議中, 以畫面分享及錄影功能, 由兩位 (含) 以上組員測試題目 3. 的 XML UI slider 控制功能。

BG3-XML UI slider 控制功能

心得：操控的程式需要到第十六週的課程網站進行下載，將程式丟入小白窗，並依照上課所學，按Tools然後Go，將控制三軸之頁面顯現，需要按To connect進行連結的動作，就可以控制 CoppeliaSim 場景的三軸。

5. 接續題目 4. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 加入 suctionPad 零件, 以手動操控或程式操控的方式, 錄製 block 方塊零件的取放模擬影片。

BG3-block 方塊零件的取放模擬

心得：將第十五週上課的程式控制方式運用在此項任務，使用鍵盤操控 K、L操作joint1，H、J操作joint2，F、G操作joint3，P、Q操作吸盤吸或放，若吸盤不小心吸到其他東西，只需要按Q就可以放開。

6. 請接續題目 5. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 以跨網路 Python remote API 的方式, 錄製圖形介面 slider 控制前三軸的轉動影片。

BG3-跨網路 Python remote API

心得：先查看協同組員之IPV4，而後將程式的self.clientID' ..... '中的IPV4改成組員的IP，組員需要在 CoppeliaSim 中的 Main script 裡增加一行程式 simRemoteApi.start(19999) ，之後與第四題動作相同，按Tools然後Go，將控制三軸之頁面顯現，需要按To connect進行連結的動作，就可以控制組員 CoppeliaSim 場景的三軸。
